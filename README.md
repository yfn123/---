# star-wish
给自己设定一些心愿清单，然后慢慢完成，让生活更加有追求、有盼头
这是一个盛放心愿的玻璃瓶。每一颗柔和的星星，都藏着一个待实现的约定。
✨ 堆叠星辰：星星在瓶中轻轻浮动，未完成是天青色，完成后会亮起温暖的橙光。
⏳ 时光倒计时：为心愿设一个日期，它会默默为你倒数，到期时温柔提醒。
💫 动态许愿：写下心愿的那一刻，会有一颗星星“扑通”掉进玻璃杯，充满仪式感。
愿你在这里，安放一百个闪亮的期许，等待它们一一发光。
